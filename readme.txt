Plugin adds shortcode `[unterseiten anzahl="3"]` which displays
the excerpt of the number of subpages set in parameter anzahl.

Plugin was created as help in WordPress support forums.


== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
